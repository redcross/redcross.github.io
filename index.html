Our Projects

  • Centralized login allowing our range of tools to share an authentication system so users need only one password to access all of our services.

  • iDAT, Disaster Casework System. Online/offline capable web application for completing disaster client casework in the field using an iPad, tablet, or laptop.

  • Incident Dispatch/Management System is a tool for managing the dispatch of disaster responders to incident scenes, and collecting important details and demographics for later analysis.

  • Calldown, an application that uses cloud telephony to enable mass communications to disaster responders via phone calls or text messages. Users can specify a custom message and desired number of responders for each incident, and calldown will attempt to reach responders until it gets enough positive responses (via touch-tone keypad or sms response).
